# Django Todo
Beautiful To Do List use Django!

#
## Tools
- Django 4
- Bootstrap 5

#
# How to Run Project

## Download Codes
```
git clone https://github.com/dori-dev/django-todo.git
```
```
cd django-todo
```

## Build Virtual Environment
```
python3 -m venv env
```
```
source env/bin/activate
```

## Install Project Requirements
```
pip install -r requirements.txt
```
